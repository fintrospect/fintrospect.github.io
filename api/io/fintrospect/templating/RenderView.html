RenderView

class RenderView[T] extends Filter[Request, Response, Request, View]

Used to render View objects to views based on template files. This template Filter can be added as a module filter to be applied to all routes in that module. If passed a View.Redirect instance (as is common after a PUT, POST) a redirect is generated instead.
